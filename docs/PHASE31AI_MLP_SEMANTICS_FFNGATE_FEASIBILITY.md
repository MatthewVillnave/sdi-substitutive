```

```
Note: W_low_rt = decode(.sdiw packed W_low)

### C. Partial Substitutive MLP (W_low + R for gate, exact up/down)
```
W_gate_sub = W_low_rt + R_sparse
Y = SiLU(X @ W_gate_sub.T) * (X @ W_up.T) @ W_down.T
```
Note: R_sparse = decode(.sdir)

### D. Strict Substitutive MLP (gate artifacts only, up/down exact)
